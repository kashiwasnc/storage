# [Vue Quiz](./index.html)

## Overview

Vue.js を用いた、ウェブブラウザ上で動作する簡単なクイズアプリです

## Requirement

- Windows 10
  - Visual Studio Code

## Description

1. YES/NO の二択問題が５問続けて出題されます
1. 回答した内容は Vue インスタンス内に保存されます
1. 全問に回答した後、結果を表示します
1. 結果をクリアして最初からクイズをやり直せます
