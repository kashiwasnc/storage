# [Currency Calculator](./index.html)

## Overview

ウェブブラウザ上にて二種類の通貨間で両替を行います  
為替レートデータは [Exchange Rate API](https://www.exchangerate-api.com/) から取得しています

## Requirement

- Windows 10
  - Visual Studio Code

## Description

1. 初期状態として、1 米ドルあたりの日本円の金額を表示させています
1. 通貨もしくは金額を変更すると自動的に再計算します
